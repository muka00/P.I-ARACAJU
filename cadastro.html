Termos de Uso e Politica de Privacidade

1. Termos de Uso

Ao utilizar os servicos da Prefeitura Municipal de Aracaju, voce concorda com os seguintes termos:

1.1. Uso do Sistema

O usuario concorda em utilizar o sistema apenas para fins legais e de acordo com a legislacao brasileira.

1.2. Responsabilidades

O usuario e responsavel por manter a confidencialidade de sua senha e por todas as atividades que ocorram em sua conta.

1.3. Conduta do Usuario

E proibido:

  • Fornecer informacoes falsas ou enganosas
  • Utilizar o sistema para atividades ilegais
  • Tentar acessar dados de outros usuarios
  • Realizar acoes que possam prejudicar o sistema

2. Politica de Privacidade

A Prefeitura Municipal de Aracaju esta comprometida com a protecao de seus dados pessoais.

2.1. Coleta de Dados

Coletamos os seguintes dados pessoais:

  • Nome completo
  • CPF
  • E-mail
  • Telefone
  • Endereco (para moradores)
  • Informacoes de localizacao (para visitantes)

2.2. Uso dos Dados

Seus dados sao utilizados para:

  • Prestacao de servicos municipais
  • Comunicacao oficial
  • Melhoria dos servicos
  • Estatisticas e planejamento urbano
  • Atendimento ao cidadao

2.3. Compartilhamento de Dados

Seus dados poderao ser compartilhados com:

  • Orgaos publicos quando exigido por lei
  • Prestadores de servicos credenciados
  • Secretarias municipais para prestacao de servicos

2.4. Seguranca dos Dados

Implementamos medidas de seguranca para proteger seus dados contra acesso nao autorizado, alteracao, divulgacao ou destruicao.

2.5. Direitos do Usuario

Voce tem direito a:

  • Acessar seus dados pessoais
  • Corrigir dados incorretos
  • Solicitar a exclusao de dados
  • Revogar consentimento
  • Solicitar portabilidade de dados

2.6. Retencao de Dados

Mantemos seus dados pelo tempo necessario para cumprir as finalidades para as quais foram coletados, conforme exigido por lei.

2.7. Contato

Para exercer seus direitos ou esclarecer duvidas sobre privacidade, entre em contato:

  • E-mail: privacidade@aracaju.se.gov.br
  • Telefone: (79) 3179-1000
  • Endereco: Rua Santa Luzia, 795 - Centro, Aracaju/SE

2.8. Alteracoes na Politica

Esta politica pode ser atualizada periodicamente. Alteracoes significativas serao comunicadas aos usuarios.
